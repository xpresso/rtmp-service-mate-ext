Extension for Mate Flex framework which collaborate with rtmp servers (Flash Media Server, Wowza, Red5)
similar as Mate and Flex collaborate with web or http services.

See quick start guide http://code.google.com/p/rtmp-service-mate-ext/wiki/quick_start

To build red5-example module with maven you need to install red5-example/libs/red5.jar
to local repository. That can be done with red5-example/libs/install.sh.

To build wowza-example module with maven you need to install wowza-example/libs/wms-core.jar and
wowza-example/libs/wms-server.jar to local repository. That can be done with
wowza-example/libs/install.sh
